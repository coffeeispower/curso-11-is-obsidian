## Exercícios sobre [[Polinómios]]

### Adição

$$
\displaylines {
(7x+3)+(2y-4x+4)=\\
=7x+3+2y-4x+4=\\
=3+4+7x-4x+2y=\\
=7+3x+2y
}
$$

### Subtração
$$
\displaylines {
(7x)-(2-4x)\\
=7x-2+4x\\
=-2+11x
}
$$
### Multiplicação
$$
\displaylines {
(4x-3)\times(2x-1)=\\
=8x^2-10x + 3\\
}
$$

### Considera os polinómios

$A(x)=x^2-4x$
$B(x)=x^2-4$
$C(x)=-x-1$
Calcula
- $A(x)-C(x)$
	- $x^2 - 4x + x + 1 = x^2-3x+1$
- $(A(x))^2+B(x)$
	- $(x^2-4x)^2 + (x^2-4)^2 = x^4 - 4x^2 + x^4-16 = 2x^4-4x^2-16$
- $A(x) \times B(x)-2(C(x)+3)$
	- $(x^2-4x)(x^2-4)-2(-x-1+3)$
	- $(x^2-4x)(x^2-4)-2(2-x)$
	- $(x^2-4x)(x^2-4) -4 + 2x$
	- $x^4 - 4x^2 - 4x^3 + 16x - 4 + 2x$
	- $x^4 - 4x^2 - 4x^3 + 18x - 4$
