## Sinal

| x     | 0   |     | $\frac{\pi}{2}$ |     | $\pi$ |     | $\frac{3}{2}$ |     | $2\pi$ |

## Injetiva
A função não é injetiva pois $\sin(\pi)=\sin(2\pi)=0$.
