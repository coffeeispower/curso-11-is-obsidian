A Escala de borg é a escala que mede o nível de esforço sentido pelo indivíduo que está a praticar o exercício.

| Valor na Escala de Borg | Legenda            |
| ----------------------- | ------------------ |
| 0,5                     | mínimo             |
| 1                       | muito fraco        |
| 2                       | fraco              |
| 3                       | moderado           |
| 4                       |                    |
| 5                       | forte              |
| 6                       |                    |
| 7                       | muito forte        |
| 8                       |                    |
| 9                       |                    |
| 10                      | extremamente forte |
| 11                      | máxima             |
